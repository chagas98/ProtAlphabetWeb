Grupo de Design de Proteínas

Você sabia que os mesmos modelos de Inteligência Artificial usados para gerar imagens a partir de texto também podem ser empregados para criar novas estruturas de proteínas?

Eles são chamados modelos de difusão!

Em modelos de difusão, uma imagem, por exemplo, é gradualmente destruída pela adição de ruído até se tornar totalmente ruidosa. Esse processo gera milhares de imagens com diferentes niveis de ruído, que são utilizadas para treinar uma Inteligência Artificial capaz de prever qual seriam imagens reais a partir de uma versão com ruído. Assim, a IA aprende a gerar novas imagens.
